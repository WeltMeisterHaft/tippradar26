# Kurzanleitung fuer Familien

1. Ein Erwachsener meldet sich mit seiner E-Mail-Adresse an.
2. Beim Beitritt `Family` waehlen.
3. Im Kontofenster ueber `+ Kinderprofil` alle teilnehmenden Kinder anlegen.
4. Vor dem Tippen immer bei `Wer tippt gerade?` das richtige Profil auswaehlen.
5. Fuer jedes Profil Tipps und eine eigene Top-5-Auswahl speichern.

Kinder benoetigen weder eine eigene E-Mail-Adresse noch ein eigenes Geraet.
